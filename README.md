# BibliotecaConsoleApp

Sistema de gerenciamento de biblioteca desenvolvido em C# (Console App), com arquitetura em camadas e persistência de dados em arquivos JSON.

## Funcionalidades

- Cadastro de livros
- Consulta de livros
- Remoção de livros
- Cadastro de usuários
- Controle de empréstimos e devoluções
- Histórico de operações (em breve)

## Tecnologias e Conceitos Utilizados

- C# (.NET)
- Programação Orientada a Objetos (POO)
- Arquitetura em camadas
- Persistência com JSON
- Git e GitHub
